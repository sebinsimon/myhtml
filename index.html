We can use wireframe to access the webpage from any devices with small, medium and wider screen size. With wireframe when we use from small device it shows in single column, if its medium screen size device then it uses two column layout and if its a wider screen size then it will use three column size. Its up to us we can use different layout like four column or five column style. In order to make my webpage more user friendly I used <meta> tag inside my <head> tag so that a mobile browser can behave like a desktop browser and display the content in full size.

For example I used the following property in my index.html page 
<meta name="viewport" content="width=device-width, initial-scale=1">

Another CSS style is grid layout which offers a grid-based layout system, with rows and columns, it makes easier to design a web page without having to use floats and positioning (CSS Grid). Then I used media queries, which is a CSS element which help users to view in horizontal, small, medium and wider screen devices using the display:line in the media queries. We use media queries because most users access website from mobile device and if we do not use a media query it wont display properly on small devices and some images look larger which is not a user-friendly website that is why we use media query which will automatically change the layout to the preferred size for that particular device.

Below is an example code for small screen size devices: 
#wrapper  {
  display: grid;
  grid-gap: 20px;
  grid-template-areas:
      "banner"
      "menu"
      "nav"
      "table"
      "main"
      "aside"
      "footer";
  }
For medium screen size layout I have used following CSS media query: 
@media (min-width: 50em)  {
  #wrapper {
    grid-template-columns: 3fr 6fr;
    grid-template-areas:
      "banner  banner"
      "menu  menu"
      "nav  table"
      "aside  main"
      "footer  footer";
    }
}
In order to support on a wider screen size we have to use a 3-column layout as I used on my CSS. I have used following media query in my CSS: 
@media  (min-width: 60em)  {
  #wrapper  {
     grid-template-columns: 3fr 5fr 3fr;
     grid-template-areas:
       "banner banner banner"
       "menu menu menu"
       "nav table aside"
       "nav main aside"
       "footer footer footer";
       }
    }
It is important to make our image flexible because when we resize the browser it automatically adjust to the size of the browser. When we create a CSS stylesheet it is important to lookout for a prefect layout and it should work on different screen size. A flexbox and grid play a major role when we style them, at first it is very difficult to understand the complex of CSS but eventually we familiarise with most properties in CSS (Grids layout). We need to use the following element in order to make the image flexible:

Today I have learned about CSS3 flexbox and multi-columns. There are four layout modes in flexbox which is, block (for sections in a webpage), Inline (for text), Table (for two-dimensional table data), and Positioned (for explicit position of an element). Multi-column layout allows easy definition of multiple columns of text which is like newspaper style (CSS Multi-column Layout). This include number of columns in a layout, as well as how content should flow from column to column, gap size between columns, and columns dividing lines along with their appearance (CSS Multi-column Layout). Flexbox is used to layout one-dimensional method for laying out items in rows or columns. Items flex to fill the spave and shrink to fit for smaller screen. Flex layout one is list items which is the navigation for a site. It is laid out as a row, with an equal amount of space between each items. Flex layout two is a list items which have different sizes, but we can display them on three equal size columns even if the content is small or large. In HTML we can use two elements .parent is a div with class and it contains another div with a class of .child (Flexbox Layout). Below is an example of parent and child div class in CSS 

 .parent {
 }
Below is an example of parent and child div class in HTML 
<div class="parent">
<div class="child">Center me.</div>
</div>

CSS Grid layout is a two-dimensional layout system for the web. It helps to lay the content in rows and columns, and has many features to make a complex layouts. A grid will typically have columns, rows, and then gaps between each row and column, it is known as gutters (Grids). Floats property is commonly used tools for creating multiple column layouts on webpages. With float property we can use the image to float left or right to move the image to that side. Positioning is another property which allows to take elements out of the normal document layout and make them behave differently (Positioning).
